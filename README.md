# 🎓 Student Management System

## 📌 Project Description
This is a Python-based Student Management System built using Object-Oriented Programming (OOP), File Handling, and Data Analytics.  
It helps manage student records like Add, View, Search, Delete, Update, and perform basic analysis on student data.

---

## 🔄 Project Flow
1. User runs the program (`student.py`)
2. Main menu is displayed
3. User selects an option:
   - Add Student → store student details
   - View Students → display all records
   - Search Student → find specific student
   - Delete Student → remove student
   - Update Student → modify details
   - Analytics → topper, average, sort, pass/fail report
4. All data is saved in `students.txt`
5. Program continues until Exit is selected
